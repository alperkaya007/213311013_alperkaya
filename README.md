# Mobile App Project — Flutter

A Flutter mobile application developed as part of my mobile development learning journey. Built with Flutter and Dart, targeting both iOS and Android platforms.

## Tech Stack

- **Flutter** 3.x
- **Dart** 3.x
- **Swift** (iOS native modules)
- **Kotlin** (Android native modules)

## Features

- Cross-platform: runs on iOS and Android from a single codebase
- Clean and minimal UI
- Responsive layout

## Getting Started

### Prerequisites

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
- Xcode (for iOS)
- Android Studio (for Android)

### Run the app

```bash
flutter pub get
flutter run
```

## Project Structure

  main.dart

  widgets/
